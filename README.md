# node.js + express + angular  + mongoDB

This project requires [Node.js](https://nodejs.org/) v8.9.4+ to run. 


### For local development

Run with [nodemon](https://nodemon.io/)
```sh
$ cd project_folder
$ npm install
$ nodemon index.js
```

### For production

```sh
$ cd project_folder
$ npm install
$ npm start
```
